# ✨ About

**Fanlar Olami** — maktab o'quvchilari va abituriyentlar uchun yaratilgan interaktiv ta'lim platformasi.

Platforma formulalarni o'rganish, test ishlash va fanlarga oid ma'lumotlarni qulay tarzda ko'rish imkonini beradi.
