# 🎓 EduVerseX

EduVerseX — HTML, CSS va JavaScript yordamida yaratilgan zamonaviy ta'lim platformasi. Loyiha o'quvchilarga fanlarni o'rganish, testlar ishlash va ta'limiy resurslardan qulay foydalanish imkonini beradi.

## ✨ Imkoniyatlari

- 📚 Turli fanlarni o'rganish
- 🔎 Qidiruv tizimi
- 📝 Interaktiv testlar
- 📖 Elektron kutubxona
- 🗺️ Xaritalar bo'limi
- 🎮 Ta'limiy mini o'yinlar
- 🏆 Leaderboard
- 💬 Chat tizimi
- 👤 Login va ro'yxatdan o'tish
- 📱 Responsive dizayn

## 🛠️ Texnologiyalar

- CSS3
